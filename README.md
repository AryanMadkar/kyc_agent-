# KYC Processing API Server

A production-ready Flask server for KYC document processing using AI models.

## Features

- Parallel document processing with multiple workers
- Support for file uploads and URL-based processing
- Automatic retry logic
- Comprehensive error handling
- Health checks and system monitoring
- Production-ready with Gunicorn
- Containerized with Docker
- Ready for Render deployment

## Quick Deploy to Render

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/yourusername/kyc-processing-api)
